# Keystone Water Initiative — GitHub Pages Homepage

This is a static homepage for GitHub Pages.

## Files
- `index.html`
- `styles.css`
- `assets/logo.png`

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save.
7. GitHub will publish the site at your Pages URL.

## Custom Domain
Once GitHub Pages is live:
1. Go to **Settings → Pages**.
2. Add your custom domain.
3. Update DNS at your registrar to point to GitHub Pages.
